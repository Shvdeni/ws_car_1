Package: touch_car_control

What it is for:
ROS 2 Python control node for the car. It drives forward, listens to the
Gazebo bumper contact topic, backs up about 5 cm after a touch, turns by a
random angle, and then drives forward again. It supports the Gazebo Sim
ros_gz contact message used by ROS 2 Jazzy. It also uses odometry as a
simulation fallback when Gazebo does not emit contact messages for a spawned
model.

Useful commands:
  source /opt/ros/jazzy/setup.bash
  colcon build --packages-select touch_car_control
  source install/setup.bash
  ros2 run touch_car_control bumper_driver
