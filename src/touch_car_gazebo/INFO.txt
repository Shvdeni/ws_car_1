Package: touch_car_gazebo

What it is for:
Gazebo Sim package. It starts a 1.5 x 1.5 meter room world, spawns the camera
car, adds a visual red sphere target, bridges the command/odometry/contact/camera
topics, bridges the SetEntityPose service used for sphere respawn, and starts
the controller plus camera viewer.

Dependency note:
  This launch uses Gazebo Sim through ros_gz.
  On ROS 2 Jazzy install it with:
    sudo apt update && sudo apt install ros-jazzy-ros-gz ros-jazzy-cv-bridge

Useful commands:
  source /opt/ros/jazzy/setup.bash
  colcon build
  source install/setup.bash
  ros2 launch touch_car_gazebo room.launch.py
